# AI & BioTech Forum 2026

Privacy-Preserving AI Infrastructure: A Federated Learning Toolkit for Cross-Institutional Collaboration

## Deploy to Streamlit Cloud

### Step 1: Push to GitHub

```bash
git init
git add .
git commit -m "AI & BioTech Forum 2026"
git remote add origin https://github.com/dechang64/ai-biotech-forum.git
git push -u origin main
```

### Step 2: Connect to Streamlit Cloud

1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Click "New app"
3. Select repository: `dechang64/ai-biotech-forum`
4. Branch: `main`
5. Main file path: `app.py`
6. Click "Deploy"

### Step 3: Access

Your app will be live at: `https://ai-biotech-forum.streamlit.app`
